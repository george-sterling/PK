Great Fern Tree

Great Fern with Sap Great Fern without Sap


By tapping this tree, and selecting "Harvest Great Fern Sap", you will receive Great Fern Sap which is an essential ingredient for crafting many different items. The Fern Sap grows naturally on all Great Fern Trees, but it can only be collected once per tree every six hours.

Great Fern Trees are also useful for Wood. If you can resist chopping them down for Wood, the ferns will multiply in a cluster over time. If you choose to chop the fern, don't fret because Great Fern Trees will regenerate randomly throughout Parallel Kingdom.
